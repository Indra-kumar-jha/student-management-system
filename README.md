Student Management System

A full-stack web-based Student Management System built using Flask.  
This application allows administrators to manage student records and students to securely access their profiles.


 Features

 Admin Panel
- Secure Admin Login
- Add New Students
- Edit Student Details
- Delete Students
- Auto-generate Username & Password for Students
- Role-Based Access Control

 Student Panel
- Secure Student Login
- View Own Profile Only
- Change Password
- Automatic Logout on Browser Close

---

 Tech Stack

- **Backend:** Flask (Python)
- **Frontend:** HTML, CSS, JavaScript
- **Database:** SQLite
- **Authentication:** Flask-Login
- **Deployment:** Render
- **WSGI Server:** Gunicorn
